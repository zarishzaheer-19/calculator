# 🌸 Cute Theme Calculator

A cute and colorful calculator built using **HTML**, **CSS**, and **JavaScript**. This calculator performs basic arithmetic operations and allows users to switch between different color themes for a fun, personalized experience.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🗑️ Clear and Delete buttons
- 🎨 Theme changer (Pink, Purple, Blue, Mint, and Dark)
- 💖 Cute heart-shaped buttons
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
